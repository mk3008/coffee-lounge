# chat package (coffee-lounge)

このパッケージは coffee-lounge モノレポにおける **中心機能** である。

Phase 1 (MVP) では、以下のみを責務とする。

## 役割

- チャットによる会話ループを提供する
- 一人壁打ちのための対話を成立させる
- 思考を整理するための出力を返す

## Phase 1 のスコープ

含むもの:
- 入力 → 応答の会話ループ
- スレッド単位の会話（1人前提）
- 人格・規約に基づいた返答

含まないもの:
- 記憶の分類・タグ付け
- 知識マップ
- マルチユーザ対応
- UI の作り込み
- 上流工程オートマトン機能

## 設計上の注意

- 「賢さ」よりも「成立」を優先する
- すべてを常時プロンプトに投げない
- 将来の拡張のために責務を増やさない

このパッケージは、coffee-lounge の足場であり、
完成形ではない。
